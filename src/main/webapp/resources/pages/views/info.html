Тестовое задание

Создать Тестовый сайт с информацией о пользователях.
На сайте должна быть авторизация, страница информации о пользователе, страница статуса пользователя и страница где он их меняет (Offline and Online статус).

Требование к сайту:

  1. Использовать Базу данных
  2. Использовать Ajax (хотя бы на одной странице)
  3. Язык программирование не важен.

Дополнительно:

  1. использовать чистый Ajax без Фреймворков типа JQuery, т.е. только средствами Javascript.
  2. Кроме статуса, отображать когда пользователь был в Сети(Online) в последний раз.
  3. Страница регистрации пользователя с проверкой правильности ввода.
  4. Можно что-то добавить по своему усмотрению. Совет: подумать, что нахватает на сайте.

Каждый дополнительный пункт это +.
